Copilot para Microsoft 365 es uno de los "Copilotos" de Microsoft a los
que, si el caso de negocio lo determina, se le puede sacar jugo más
rápidamente gracias a su integración completa con muchos servicios y
herramientas de Microsoft 365 como Microsoft Teams, Microsoft Loop,
Microsoft Forms, etc. Pero: ¿Cómo podemos saber al menos de manera
inicial como de preparados estamos para la llegada de Copilot para
Microsoft 365? La respuesta es que a través Copilot for Microsoft 365
Optimization Assessment provisto por Microsoft. En este artículo vamos a
revisar el Assessment y los reports que genera.

**Usando el Copilot for Microsoft 365 Optimization Assessment**

Para hacer uso del assessment, tenéis que acceder a la siguiente URL:
[Solution Assessment Program
(microsoft.com)](https://www.microsoft.com/en-us/solutionassessments/safeedbackform).
Como véis, el assessment está pensado para que lo realicen partners de
Microsoft para sus clientes o directamente esto.

![A screenshot of a computer error Description automatically
generated](../../../images/numero59/copilot-for-m365-solution-assessment/image1.png)

Tras elegir la opción para responder el assessment, a continuación
tendremos que especificar una serie de datos antes de comenzar con el
cuestionario que está organizado por una serie de secciones.

![A screenshot of a computer Description automatically
generated](../../../images/numero59/copilot-for-m365-solution-assessment/image2.png)

Como veis, las secciones en las que se divide el Assessment son las
siguientes:

-   Optimización de usuarios, en la que se realizan cuestiones relativas
    al estado actual de despliegue de Microsoft 365 en la organización
    en cuanto a número de usuarios, licenciamiento disponible, etc.
    Tened en cuenta los requerimientos de licenciamiento de Copilot para
    Microsoft 365 para entender esta sección. [Microsoft Copilot for
    Microsoft 365 requirements | Microsoft
    Learn](https://learn.microsoft.com/en-us/microsoft-365-copilot/microsoft-365-copilot-requirements).

![A screenshot of a computer checklist Description automatically
generated](../../../images/numero59/copilot-for-m365-solution-assessment/image3.png)

-   Perfil de organización, en la que se realizan cuestiones relativas
    al nivel de madurez de la organización para la llegada de la
    Inteligencia Artificial con soluciones como Copilot para Microsoft
    365 u otras.

![A screenshot of a computer Description automatically
generated](../../../images/numero59/copilot-for-m365-solution-assessment/image4.png)

-   Herramientas de Productividad y Colaboración en la que se realizan
    cuestiones a las herramientas y servicios de productividad en uso.
    También se realizan cuestiones relativas a los orígenes de
    información principales de la organización que se tienen que
    considerar para el despliegue de Copilot. Las preguntas de esta
    sección son particularmente importantes dado que dependiendo del
    estado de uso y adopción de Microsoft 365, se le podrá sacar un
    mayor partido a Copilot para Microsoft 365.

![A screenshot of a computer Description automatically
generated](../../../images/numero59/copilot-for-m365-solution-assessment/image5.png)

-   Seguridad del Dato y Gobierno, sección en la que se realizan
    cuestiones relativas al estado del arte en lo que a compartir
    información, proteger información, detectar fugas de información
    (DLP) o retener información se refiere. Como podréis suponer, es en
    esta sección donde muchas organizaciones tienen mucho trabajo por
    hacer.

![A screenshot of a computer error Description automatically
generated](../../../images/numero59/copilot-for-m365-solution-assessment/image6.png)

Una vez contestamos a las 25 preguntas del Assessment, podemos enviar el
mismo y obtener un resultado preliminar por pantalla de nuestro nivel de
madurez para la adopción de Copilot para Microsoft 365.

![A screenshot of a computer Description automatically
generated](../../../images/numero59/copilot-for-m365-solution-assessment/image7.png)

**Resultados del Assessment**

Como habéis visto en la última captura del Assessment, tras enviar el
cuestionario se muestra nuestro nivel de madurez en tres categorías:

-   Perfil de organización.

-   Uso de herramientas de productividad.

-   Securización del dato.

Los % que aparecen se pueden interpretar con la ayuda provista en el
propio Assessment y disponible en este enlace: [Read me first.pdf
(windows.net)](https://msxcuststorage.blob.core.windows.net/copilot/Read%20me%20first.pdf).
Básicamente:

-   Puntuación del 0% - 33%: La organización puede obtener más con
    Copilot para Microsoft 365, pero requiriendo cierto esfuerzo de
    preparación para obtener un valor máximo.

-   Puntuación del 34% - 66%: La organización puede sacar partido a las
    capacidades de Copilot para Microsoft 365. Se puede requerir trabajo
    adicional para alinear el valor de Copilot para Microsoft 365 con
    los requerimientos de la organización.

-   Puntuación del 67% - 100%: El estado actual de la organización está
    muy alineado con Coilot para Microsoft 365 y puede sacarle el máximo
    partido.

Además de los resultados de inicio, se envía un documento PDF con:

-   Los resultados globales del Assessment.

-   Las respuestas a las preguntas planteadas.

-   Recomendaciones y enlaces informativos para mejorar la preparación
    para la llegada de Copilot para Microsoft 365.

![A screenshot of a computer Description automatically
generated](../../../images/numero59/copilot-for-m365-solution-assessment/image8.png)

**Conclusiones**

Sin duda, toda ayuda para determinar el grado de preparación de una
organización para Copilot para Microsoft 365 es más que bienvenida. En
este sentido, Microsoft pone a nuestra disposición el Copilot for
Microsoft 365 Optimization Assessment para darnos una medida de dicha
preparación.
